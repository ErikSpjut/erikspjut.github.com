This folder contains example files from a project that Mary Cardenas and Erik Spjut worked on in the Summer and Fall of 1998. The intent of the project was to provide solutions to problems in Felder and Rousseau, 2nd Edition, which presented both solution techniques and software tutorials. The problem numbers correspond to the numbers in F&R 2nd. Ed., but the problem statements are included in the .pdf files, so a copy of F&R is not needed.
